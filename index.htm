Website sits on the internet, which is the interaction of computer networks. It's a wire connecting several computers.Internet connection is like the wire connecting the clients to the severs.
Browsers are softwares that interprete or renders webpages. Clients are the typical web-users intrernet-conneced devices e.g.your PC. The internet and browser is what a client machine needs.

How a Website Works

When you type a web address i.e URL(UNIFORM RESOURCE LOCATOR) into your browser,the URl will be sent to an Internet Service Provider(ISP), which will then direct the URL to the DNS(Domain Name System) server-dedicated servers that are like address book for websites, always running 24/7. The DNS goes through its database and finds a corresponding adress to the URl. this adress is known as IP address-Internet protocol.

Protocol is a set of rules that establishes inter-machine relation.There are differnt types of protocols depending on what you want to use it for.e.g Transmision Control Protocol(TCP)

A website can be made using 3 different languages:

  1. HTML-Hyper Text Mark-up Language:For making the webite contents.It takes care of the layout of the website.
  2. CSS-Cascading Style Sheet:For styling the website.
  3. Java Script:For making the website interactive.It's takes care of the functionality of the website.

NOTE:Html can conveniently make a website unlike CSS and Java script
